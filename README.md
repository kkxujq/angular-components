## 安装使用

```bash
npm set registry=http://192.168.10.223:7001/
npm install get-ui-ng --save
```

或者
```bash
yarn config set registry http://192.168.10.223:7001/
yarn add get-ui-ng
```

## 本地build
```bash
npm run build
```

## 运行Demo服务

```bash
npm run demo-app
```

## 运行文档服务
```bash
npm run doc-app
```
