GtSubMenu

次级菜单定义

Properties
Name Description

显示选中状态的url链接，解决二级子菜单还有子菜单时link不匹配的问题

label: string

次级菜单展示名

link: string

菜单的url链接

name: string

次级菜单名称（暂时无用）

selected: boolean

是否选中
